# Resume Project

A responsive, professional resume website built with HTML and CSS.

## Project Structure

```
resume/
├── index.html       # Main resume page
├── styles.css       # Styling and layout
├── images/          # Image assets
└── README.md        # This file
```

## Features

- Clean, modern design
- Responsive layout
- Professional presentation

## Getting Started

1. Open `index.html` in your web browser
2. View your resume in a beautiful, formatted layout

## Customization

Edit `index.html` to add your personal information, experience, and skills.
Modify `styles.css` to customize colors, fonts, and layout.

## Files

- **index.html** - Main HTML structure
- **styles.css** - CSS styling
- **images/** - Directory for profile images and other assets
